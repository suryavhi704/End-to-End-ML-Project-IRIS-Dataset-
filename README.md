# 🌸 End-to-End Machine Learning Project on Iris Dataset  

## 📖 Project Overview  
This project demonstrates an **End-to-End Machine Learning workflow** on the famous **Iris dataset** using **Scikit-learn, FastAPI, and Uvicorn**.  
It covers the full ML pipeline from **data exploration to deployment**, making it a complete hands-on project for real-world applications.  

---

## ⚡ Workflow  

1. **Data Loading & Understanding**  
   - Loaded the Iris dataset from Scikit-learn.  
   - Studied dataset structure (features & classes).  

2. **Exploratory Data Analysis (EDA)**  
   - Visualized distributions and pairplots.  
   - Checked correlations between features.  
   - Identified patterns in species classification.  

3. **Data Preprocessing**  
   - Normalized feature values.  
   - Handled data splits (train/test).  

4. **Model Selection & Training**  
   - Compared multiple ML models (Logistic Regression, KNN, SVM, Decision Tree, Random Forest).  
   - Selected the **best-performing model** based on accuracy.  

5. **Model Saving**  
   - Saved the trained model using **Pickle** for later use.  

6. **Model Deployment**  
   - Built an **API using FastAPI**.  
   - Served predictions with **Uvicorn server**.  

---

## 🛠️ Tech Stack  

- **Language:** Python  
- **Libraries:** Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn  
- **Deployment:** FastAPI, Uvicorn  
- **Serialization:** Pickle  

---
## 📊 Results & Insights

Achieved high accuracy (>95%) on classification.

Visualized clear separability of Iris species.

Created a lightweight ML API that can be integrated with web or mobile apps.
